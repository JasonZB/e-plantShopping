# Paradise Nursery Shopping Application 🌱

Este proyecto es el **Final Project** del curso de Coursera.  
La aplicación simula una tienda en línea de plantas de interior, permitiendo a los usuarios:

- Explorar diferentes categorías de plantas.
- Agregar productos al carrito de compras.
- Visualizar el total de ítems y costos.
- Administrar cantidades, eliminar productos y ver subtotales.
- Navegar entre páginas con un **navbar** dinámico.

## Tecnologías utilizadas
- React con Vite
- Redux Toolkit para el manejo de estado global
- CSS para estilos y diseño responsivo

## Estructura principal del proyecto
- `src/CartSlice.js` → Slice de Redux para manejar el carrito.
- `src/store.js` → Configuración del store global.
- `src/main.jsx` → Conexión del store con la aplicación.
- `src/ProductList.jsx` → Listado de productos y botón “Add to Cart”.
- `src/CartItem.jsx` → Página del carrito con funcionalidades completas.
- `src/App.jsx` → Landing page con nombre de la compañía y botón “Get Started”.
- `src/AboutUs.jsx` → Página con detalles de la compañía.
